AIGS - Javadoc compilation

This document contains links to several Javadoc resources of AIGS. The Javadoc resources were compiled per project and collected together.
The following resources are available. Please click on the links below to jump directly to the particular resource.

Project Type Link Description
AIGS BaseClient Library ./javadoc/AIGS_BaseClient/index.html The AIGS BaseClient is the library for all JavaFX based games in AIGS. The project provides AIGS_BaseClient.jar
AIGS Commons Library ./javadoc/AIGS_Commons/index.html AIGS Commons is the shared library for the AIGS client and server components. The project provides AIGS_Commons.jar
AIGS Server Application ./javadoc/AIGS_Server/index.html AIGS Server is the server application of AIGS
AIGS SwingBaseClient Library ./javadoc/AIGS_SwingBaseClient/index.html The AIGS SwingBaseClient is the library for all Swing based games in AIGS. The project provides AIGS_SwingBaseClient.jar
BinaerOperatoren Game ./javadoc/BinaerOperatoren/index.html BinaerOperatoren is one of the prepared games of AIGS. It is based on Swing
Minesweeper Game ./javadoc/Minesweeper/index.html Minesweeper is one of the prepared games of AIGS. It is based on JavaFX
RockPaperScissors Game ./javadoc/RockPaperScissors/index.html RockPaperScissors is a tutorial game. It is based on JavaFX. Javadoc is written in German
TicTacToe Game ./javadoc/TicTacToe/index.html TicTacToe is one of the prepared games of AIGS. It is based on JavaFX
